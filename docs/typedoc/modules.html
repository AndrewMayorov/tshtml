transformAttrs

  • transformAttrs(args: Dictionary<any>, mapping?: Dictionary<string | TransformArgumentMapping>): Dictionary<any>
    • Defined in transformAttrs.ts:34

    Transform arguments object into another arguments object renaming arguments along the way (optionally) and applying transformations (optionally). If an arguments is not specified in the mapping object, then it is copied as-is. Example call:

    this.transform(
    {
        "href": "/uhus.html",
        "ngIf": "ctl.isNew"
    },
    {
        ngIf: "ng-if",
        ngShow: { to: "ng-show", default: false }
    } ) );

    Here we have argument href passed as-is, ngIf gets converted to ng-if, and ng-show is added because it has default value specified.

    Optional transform parameter allows for transform argument value during mapping. Default values gets transformed too.

    Parameters

    • args: Dictionary<any>

      Original arguments object

    • Optional mapping: Dictionary<string | TransformArgumentMapping>

      Mapping configuration

    Returns Dictionary<any>
